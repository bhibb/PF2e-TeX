Using the Pathfinder-Icons Font

The Pathfinder-Icons.ttf font includes the symbols used to denote different actions in the Pathfinder Roleplaying Game.

How to Use

Install the font as directed for your operating sytem and/or application. Your application must support ligatures to work properly.

To display a symbol, set the corresponding text exactly as listed below, including the brackets, and set that text to use the Pathfinder-Icons font.

SymbolAction TypeText
Single Action[one-action]
Two-Action Activity[two-actions]
Three-Action Activity[three-actions]
Reaction[reaction]
Free Action[free-action]
